## 依赖

### Maven

```` xml
<dependency>
   <groupId>cn.chenzw.toolkit</groupId>
   <artifactId>toolkit</artifactId>
   <version>1.0.2</version>
</dependency>
````

### Gradle

```
compile 'cn.chenzw.toolkit:toolkit:1.0.2'
```
